Description

This is a Clothing Store Website built using HTML, CSS, and JavaScript.
The website allows users to browse clothing items and interact with them by adding products to:

 Favorites
 Shopping Cart

This project is designed for practicing frontend fundamentals and JavaScript DOM manipulation.

- Features

Display a list of clothing products

Add products to Favorites

Add products to Shopping Cart

Update cart and favorite item counts dynamically

Simple and user-friendly UI

Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)
